# ALU-X

A modular Arithmetic Logic Unit (ALU) project implemented in Verilog HDL.

The current implementation is a **4-bit ALU**, serving as the foundation for future expansion to wider datapaths, additional operations, and more advanced processor-oriented features.

---

## Current Features (v1)

### Data Width

* 4-bit operands (`A` and `B`)
* 4-bit result (`Q`)

### Supported Operations

| Opcode | Operation |
| ------ | --------- |
| `000`  | AND       |
| `001`  | OR        |
| `010`  | XOR       |
| `011`  | ADD       |
| `100`  | SUBTRACT  |

---

## Status Flags

| Flag  | Description                                                                    |
| ----- | ------------------------------------------------------------------------------ |
| `of`  | Overflow / carry flag generated by arithmetic operations                       |
| `zf`  | Zero flag, asserted when the output is zero                                    |
| `epf` | Even parity flag, asserted when the output contains an even number of logic 1s |

### Modules

#### `fa`

Single-bit full adder.

#### `addsub4bit`

4-bit ripple-carry adder/subtractor built from four full adders.

#### `alu4bit`

Top-level ALU implementing arithmetic, logical operations, and flag generation.

The project includes an exhaustive testbench that automatically evaluates every possible combination of:

## Running the Simulation

### Compile

```bash
iverilog -o alu_sim alu.v tb_alu.v
```

### Execute

```bash
vvp alu_sim
```

### View Waveforms

```bash
gtkwave alu.vcd
```
